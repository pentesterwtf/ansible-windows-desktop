* Also used for development VMs
  * Like <https://github.com/pentesterwtf/packer-windows>

## Breakdown

### Common role

* Putty
* Google Chrome
* VirtIO drivers
* Ghidra
* vscode

## Logging role

* Sysmon

## Development
